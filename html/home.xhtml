Severus of Antioch, Letter 1.35

A Sample Text Encoding Project for HIST 630: "Digital Methods for Historical Research"

This small-scale text markup project is a model for students in my digital history course. The intent is to demonstrate the application some of the skills we have learned over the course of the semester: GitHub, TEI, XQuery, XPath, ODD Customization, Schematron, HTML, and CSS.

Severus of Antioch was a Greek-speaking Christian bishop in early 6th-century Antioch. He was a proponent of non-Chalcedonian, or Miaphysite, Christology. This view emphasized the unified divine/human nature of Jesus after the incarnation. Today Syriac and Coptic Christian communities still hold this view. While we know that Severus of Antioch wrote thousands of letters, fewer than 300 remain extant, mostly in Syriac translation. This project offers an encoding of one of these letters along with some basic information about the persons and places discussed in that letter.

If you are interested in seeing the code for this project including the TEI files for the letter and the standoff markup, customized ODD schema, XQuery for transforming TEI to HTML, and CSS, you can find this material in my severus-class-project repo on GitHub.
